# Sprint 2: SaaS Landing Page

## Goal
Build a responsive, premium SaaS-style landing page for HomeNest that visualizes the core modules, displays key statistics, details the scattered data problem, provides an interactive live dashboard mockup, and guides users with clear CTAs.

## Completed Features
- [x] **Sticky responsive Navbar**: Brand logo, scrolling background blur, and collapsible mobile navigation overlay.
- [x] **Hero Section**: Two-column layout with CTA actions and a floating CSS illustration mapping dynamic cards.
- [x] **Statistics Display**: Integrated grid of household metrics (Tasks, Bills, Documents, Availability).
- [x] **Problem Map**: Visual compare flow contrasting disjointed app fatigue with the unified HomeNest experience.
- [x] **Feature Grid**: 6 hoverable card slots with transition zoom effects and matching Fi icons.
- [x] **Interactive Dashboard Mockup**: Clean React component simulating real-time check/uncheck tasks and input submissions.
- [x] **How It Works Timeline**: Step-by-step layout indicating account setup, data input, and management.
- [x] **Testimonial Slider**: Rating star cards mapping user feedback.
- [x] **CTA Banner**: Prominent block urging signups.
- [x] **Footer Layout**: Site index list mapping resources.

## Challenges
- Designing the live Dashboard Preview component to feel alive, functional, and self-contained on the landing page.

## Solutions
- Used local React state hooks (`useState`) to support interactive task management and dynamic category indicators directly inside the preview widget.
